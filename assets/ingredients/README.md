# Fotos da seção de ingredientes

Salve as fotos anexadas no chat **nesta pasta**, com exatamente estes nomes
(o `ingredients-section.html` já aponta pra eles). Enquanto não estiverem aqui,
a seção mostra um placeholder elegante — não quebra.

| Arquivo | Qual foto usar |
|---------|----------------|
| `cassava-starch.jpg`   | Tigela de polvilho (pó branco) ao lado das raízes de mandioca |
| `real-cheese.jpg`      | Queijo ralado na tigela + blocos de parmesão + ralador |
| `organic-eggs.jpg`     | Ovos quebrados na tigela de cerâmica + cartela de ovos |
| `olive-oil.jpg`        | Garrafinha de azeite + azeitonas + ramo de oliveira |
| `comfort-table.jpg`    | Pão de queijo na tigela com xícara de café (mesa de madeira) — linha "No artificial preservatives" |
| `brazilian-comfort.jpg`| Prato de pão de queijo + café na mesa rústica (luz da janela) — fechamento full-bleed |

**Formato:** `.jpg` (ou troque a extensão no HTML). Ideal ≥ 1200px de largura.
Orientação retrato funciona melhor nas linhas (proporção 4:5); a foto de
fechamento pode ser paisagem.
